# G-visual baseline screenshots

Drop three PNGs of an APPROVED build here as the visual baseline the G-visual
reviewer compares against:

- `grid.png`  - the default card grid
- `modal.png` - an opened property detail modal (e.g. openModal(1))
- `map.png`   - the map view with a POI layer + 30-min isochrone toggled

Capture them once from a build you have signed off as on-brand (the CBRE
reference look). The reviewer in `reference/visual-qa.md` reads these to judge
whether a new build matches. If absent, the reviewer judges against the CBRE
house style described in that doc.
